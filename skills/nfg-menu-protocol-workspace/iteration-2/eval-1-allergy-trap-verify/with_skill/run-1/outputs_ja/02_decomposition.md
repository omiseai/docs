# 02 · 分解 —（店名未提供 — ランチメニュー）

各料理について: 構成要素 → 材料（複合材料は展開）、調理法、全項目に出典タグ。料理ごとに
正規資産を照合したが、**どの料理・材料もシードの正規項目に一致しなかった**（メモ参照）
ため、ここに `[CANONICAL]` タグは付かない。ソースは名称＋価格のみを印字しているので、
以下のほぼすべては料理の常識からの `[INFERRED]` であり、検証が必要。

正規照合の結果（一度だけ記録）:
- `product_master.json` —— このメニューに飲料なし → 一致なし。
- `ingredient_glossary.json` —— これらの料理名/推測材料はシード用語集の語
  （タイ, カニ面, バイ貝, へしこ, 片栗粉, 能登牛, 若狭牛）に該当しない。`[CANONICAL]`
  名は適用なし。（衣に 片栗粉/potato starch を使うと判明すれば用語集項目が適用される ——
  ただしそれは OPEN gap であって確定事項ではない。）
- `food_culture_kb.json` —— これらの料理は KB になし（あるのは へしこ, 柿の葉寿司 のみ）。
  よって文化的ナラティブは `[INFERRED]` であり、決して `[CANONICAL]` ではない。

## 料理1 — ロースかつ定食（Pork loin cutlet set meal [INFERRED·high]）
- 英語名: Pork loin cutlet (tonkatsu) set meal  [INFERRED·high]
  （用語集項目なし。ロースかつ定食 の標準的な料理上の読み）
- 構成要素・材料:
  - **カツ本体:**
    - pork loin（ロース）  [INFERRED·high]  ← 名称は豚ロースを強く示唆するが、ソースは
      「pork（豚）」を一度も印字しない。豚であること（鶏や別部位でないこと）を確認
    - 衣/バッター（複合 → 小麦粉 + 卵 + パン粉[→小麦]）  [INFERRED·med]
      ← 「かつ」＝慣習的に衣付きカツ。衣の正確な組成は未確認
      （小麦の代わりに/とともに 片栗粉/potato starch の可能性 —— 用語集参照。これは gap）
    - 揚げ油（複合 → ベース不明: 植物油？ ごま油入りブレンド？）  [UNKNOWN]
      ← 油の正体が不明。ごま/大豆/ピーナッツ油の疑問を左右する
  - **定食の付け合わせ（定食の慣習 —— ソースに印字なし）:**
    - ご飯  [INFERRED·med]
    - 味噌汁（複合 → 味噌[→大豆]、だし[→ kombu + katsuobushi/魚]、豆腐[→大豆]？、わかめ？）  [INFERRED·med]
    - 千切りキャベツ  [INFERRED·med]
    - とんかつソース（複合 → 野菜/果実 + 砂糖 + 酢 + 醤油[→大豆 + 小麦] + 香辛料）  [INFERRED·med]
    - 漬物  [UNKNOWN]
- 調理法（20種マスターより）: deep-fry  [INFERRED·high]; boil/simmer（ご飯, 味噌汁）  [INFERRED·med]
- メモ: 典型的なアレルゲンの罠 —— 小麦 + 卵 + 大豆 が衣・油・ソース・味噌汁に隠れる。
  いずれも印字なし。すべて gap になる。

- 英語名: Breaded fried shrimp (ebi-fry)  [INFERRED·high]（用語集項目なし）
- 構成要素・材料:
  - shrimp / prawn（エビ）  [INFERRED·high]  ← 名称＝エビ。印字名は種を特定しないが、
    甲殻類であること自体は名称から高信頼
  - 衣/バッター（複合 → 小麦粉 + 卵 + パン粉[→小麦]）  [INFERRED·med]
  - 揚げ油（複合 → ベース不明: 植物油？ ごま油入りブレンド？）  [UNKNOWN]
  - 添えソース/タルタルソース？（複合 → マヨ[→卵] + …）  [UNKNOWN]  ← タルタルや
    とんかつソースで提供されることが多いが、印字なし
- 調理法: deep-fry  [INFERRED·high]
- メモ: エビ＝甲殻類アレルゲンは名称からほぼ確実。*追加*のアレルゲン（小麦、卵、油）が

## 料理3 — 杏仁豆腐（Annin-dofu / almond jelly [INFERRED·high]）
- 英語名: Annin-dofu (almond-flavored jelly)  [INFERRED·high]
  （用語集項目なし。ローマ字＋注釈が適切 —— 日本/中国系のデザート。
  **「almond tofu」ではない** / **大豆の豆腐ではない** —— ここの「豆腐」は食感を表す語で
  あって大豆の豆腐ではない。これは名称の罠。）
- 構成要素・材料（典型レシピ —— 印字なし）:
  - 牛乳 and/or クリーム（乳）  [INFERRED·med]  ← 多くのレシピは牛乳ベース → 乳の gap
  - 砂糖  [INFERRED·med]
  - 凝固剤: 寒天（かんてん）**または** ゼラチン  [UNKNOWN]
    ← どちらかは不明。ゼラチンはベジタリアン/ハラル/コーシャの制限に関わる
  - アーモンド風味: 杏仁（アンズの種）抽出 **または** アーモンドエッセンス  [UNKNOWN]
    ← 本来の杏仁はアンズの種。多くの店はアーモンドエッセンスを使う。木の実（アーモンド）
      アレルゲンの有無に影響。どちらかは不明。
- 調理法: （なし / 冷やし固め）  [INFERRED·med]
- メモ:
  - 名称の「豆腐」は大豆/豆腐を含むことを意味**しない**。名称から大豆アレルゲンを
    導かないこと。（名称の罠。）
  - これはデザート —— デザートに肉を誤タグするミスを防ぐ。肉/ゼラチンの疑問は*制限*の
    問題であり、実際の材料（凝固剤）からのみ導く。決して断定しない。
