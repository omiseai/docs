# 03 · アレルゲン・制限 —（店名未提供 — ランチメニュー）

**段階02 の材料のみから機械的に**導出。各アレルゲンは記載材料まで辿れ、連鎖の中で
**もっとも弱い**出典を継ぐ。ソースは名称＋価格のみを印字したため、ほぼ全連鎖が
`[INFERRED]` か `[UNKNOWN]` の材料を通る → よってほぼすべてのアレルゲン状態は事実でなく
**GAP**。いずれも段階06 に事実として書いてはいけない。

凡例: `rec` = 推奨表示、`man` = 義務表示、`none` = その地域のリストにない、
`—` = まだ判定不能（連鎖未解決）。

## 料理1 — ロースかつ定食（Pork loin cutlet set meal）
### アレルゲン（JP / US / EU / CA / AU）
| アレルゲン | 由来材料 | 出典 | JP | US | EU | CA | AU |
|---|---|---|---|---|---|---|---|
| 小麦 | 衣の小麦粉 + パン粉 +（ソースの醤油） | [INFERRED·med] → **GAP** | man | man | man | man | man |
| 卵 | 衣のバッター | [INFERRED·med] → **GAP** | man | man | man | none | man |
| 大豆 | 味噌汁（味噌/豆腐）+ とんかつソース（醤油） | [INFERRED·med] → **GAP** | rec | man | man | man | man |
| 魚 | 味噌汁のだし → katsuobushi（かつお） | [INFERRED·med] → **GAP** | rec | man | man | man | man |
| ごま | 揚げ油のベース（ごま油入りブレンドの場合） | [UNKNOWN] → **GAP** | rec | man | man | man | man |
| ピーナッツ | 揚げ油のベース（ピーナッツ油の場合） | [UNKNOWN] → **GAP** | rec | man | man | man | man |

豚そのもの: 豚は**アレルゲンのトークンではない** —— 制限の問題（下記参照）。

### 制限（12項目マスター）
| 制限 | 状態 | 由来 | 出典 |
|---|---|---|---|
| ベジタリアン | 不可（豚を含む。だしはおそらく魚） | pork loin | [INFERRED·high] → GAP（豚を確認） |
| ヴィーガン | 不可 | 豚 + 乳(ソース?)/魚 | [INFERRED·high] |
| no-pork | **不可**（豚を含む） | pork loin | [INFERRED·high] → GAP（豚を確認） |
| ハラル | **不可**（豚。揚げ油・ソースも未確認） | pork loin | [INFERRED·high] → GAP |
| no-beef | おそらく可（牛肉は想定されない） | — | [INFERRED·low] |
| グルテンフリー | **不可**（衣の小麦） | 衣 | [INFERRED·med] → GAP |

### ここで生じた gap
- 衣の組成（小麦？卵？片栗粉？）→ 小麦 + 卵の状態
- 揚げ油のベース → ごま / ピーナッツの状態
- 味噌汁の有無と組成（*推測*の付け合わせ）→ 大豆 + 魚の状態
- とんかつソースの有無と組成 → 大豆 + 小麦の状態
- たんぱく質が豚であることを確認 → no-pork / ハラル / ベジタリアン

`[INFERRED]` を通る（ソースは「エビ」を材料リストとして印字しない）。よってこれすら
断定せずオーナー確認に回す。**隠れた**アレルゲン（小麦、卵、油）が本当の罠。

### 制限
| 制限 | 状態 | 由来 | 出典 |
|---|---|---|---|
| ペスカタリアン | 可（魚介。陸の肉は想定されない） | shrimp | [INFERRED·med] |
| ベジタリアン / ヴィーガン | 不可（甲殻類） | shrimp | [INFERRED·high] |
| グルテンフリー | **不可**（衣の小麦） | 衣 | [INFERRED·med] → GAP |
| ハラル | 未確認（油/ソース） | 揚げ油 | [UNKNOWN] → GAP |

### ここで生じた gap
- 甲殻類（えび）の確認、他の貝類/交差接触の有無
- 衣の組成 → 小麦 + 卵
- 揚げ油のベース → ごま / ピーナッツ
- 添えソース（タルタル＝卵）→ 卵

## 料理3 — 杏仁豆腐（Annin-dofu / almond jelly）
**名称の罠への注意:** 名称の「豆腐」は食感を表す語。名称から大豆アレルゲンを
導いては**いけない**。大豆は実際に大豆材料がある場合のみ導出可（想定されない）。
**デザート誤タグへの注意:** このデザートに肉/豚/牛のアレルゲンを割り当てない。
ゼラチンの疑問は凝固剤からの*制限*の問題であり、決して肉アレルゲンではない。

### アレルゲン（JP / US / EU / CA / AU）
| アレルゲン | 由来材料 | 出典 | JP | US | EU | CA | AU |
|---|---|---|---|---|---|---|---|
| 乳 | 牛乳/クリームのベース | [INFERRED·med] → **GAP** | rec | man | man | man | man |
| アーモンド（木の実） | アーモンド風味が*本物の*アーモンド/アンズの種の場合 | [UNKNOWN] → **GAP** | rec（JP: 中核リストになし） | man | man | man | man |
| 大豆 | —（なし。名称の「豆腐」は大豆ではない） | n/a | none | none | none | none | none |

### 制限
| 制限 | 状態 | 由来 | 出典 |
|---|---|---|---|
| ベジタリアン | 凝固剤次第（寒天=可 / ゼラチン=不可） | 凝固剤 | [UNKNOWN] → GAP |
| ヴィーガン | 不可（乳）+ 凝固剤 | 牛乳 | [INFERRED·med] → GAP |
| ハラル / コーシャ | ゼラチンの由来次第（豚/牛？） | 凝固剤 | [UNKNOWN] → GAP |
| 乳不使用（dairy-free） | おそらく**不可**（牛乳） | 牛乳 | [INFERRED·med] → GAP |

### ここで生じた gap
- 乳の有無（牛乳/クリーム）？ → 乳アレルゲン
- アーモンド風味＝本物のアンズの種/アーモンド vs 合成エッセンス？ → 木の実アレルゲン
- 凝固剤＝寒天 vs ゼラチン（ゼラチンなら由来）？ → ベジタリアン/ハラル/コーシャ

上記と同じ**デザート誤タグへの注意**。ゼラチン（あれば）は独立トークンであり、
