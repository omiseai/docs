# 03 · アレルゲン・制限 — そば店（`soba-shop-onboarding`）

アレルゲンは段階02 の材料から導出（決して場当たり的にタグ付けしない）。各アレルゲンは
材料まで辿れ、連鎖の中で**もっとも弱い**出典を継ぐ —— よって `[INFERRED]`/`[UNKNOWN]`
材料に触れる連鎖は事実でなく **GAP**。地域別状態: man(義務) / rec(推奨) / none。
（JP / US / EU / CA / AU。）

注: **そば（buckwheat）**は JP 義務・EU 推奨。この店の中心的アレルゲンであり、以下の
すべてのそば料理に（定義上）含まれる。

---

## 料理1 — ざるそば
### アレルゲン
| アレルゲン | 由来材料 | 出典 | JP | US | EU | CA | AU |
|---|---|---|---|---|---|---|---|
| そば | そば粉（麺） | [INFERRED·high] → ほぼ確実だがオーナー確認 | man | none | rec | none | none |
| 小麦 | (a) 麺のつなぎ小麦？（二八 vs 十割）— UNKNOWN; (b) つゆの醤油 | [INFERRED·med] / [UNKNOWN] → **GAP** | man | man | man | man | man |
| 大豆 | つゆの醤油 | [INFERRED·high] → **GAP** | rec | man | man | man | man |
| 魚 | だし → katsuobushi（かつお） | [INFERRED·med] → **GAP** | rec | man | man | man | man |

### 制限
| 制限 | 状態 | 由来 | 出典 |
|---|---|---|---|
| ベジタリアン | おそらく**不可** | つゆのだし（かつお = 魚） | [INFERRED·med] → GAP（昆布のみのだしなら可の可能性） |
| ヴィーガン | おそらく不可 | だし（魚） | [INFERRED·med] → GAP |
| ペスカタリアン | おそらく可 | 動物性はだしの魚のみ | [INFERRED·med] → GAP |
| グルテンフリー | **不可** | そばは GF だが麺/つゆの小麦は不可 | [INFERRED·med] → GAP |
| no-pork / no-beef / ハラル | おそらく可（肉なし） | 肉材料なし | [INFERRED·med]（みりんのアルコールが厳格なハラルでは不可の可能性）→ GAP |

### ここで生じた gap
- 麺の小麦配合比（二八/十割）→ 小麦アレルゲン + グルテンフリー
- つゆの組成（醤油 → 大豆+小麦; だし → 魚）→ 大豆/小麦/魚 + ベジタリアン/ヴィーガン

### 制限
| 制限 | 状態 | 由来 | 出典 |
|---|---|---|---|
| ベジタリアン / ヴィーガン | おそらく不可 | 魚だし。えび天の可能性 | [INFERRED/UNKNOWN] → GAP |
| ペスカタリアン | 天ぷらの具材次第 | えび/野菜なら可; — | [UNKNOWN] → GAP |

### ここで生じた gap

- 衣の卵含有 → 卵アレルゲン
- 揚げ油（純植物 vs ごまブレンド）→ ごまアレルゲン
- つゆの醤油 + だし → 大豆/小麦/魚
- 麺の小麦配合比 → 小麦

## 料理4 — そばがき
### アレルゲン
| アレルゲン | 由来材料 | 出典 | JP | US | EU | CA | AU |
|---|---|---|---|---|---|---|---|
| そば | そば粉 | [INFERRED·high] → 確認 | man | none | rec | none | none |
| 小麦 | つなぎ小麦を加える場合のみ（通常なし）または 添えのつゆの醤油 | [INFERRED·med] / [UNKNOWN] → **GAP** | man | man | man | man | man |
| 大豆 | 添えのつゆの醤油（つけ汁で提供する場合） | [INFERRED·med] → **GAP** | rec | man | man | man | man |
| 魚 | 添えのつゆのだし | [INFERRED·med] → **GAP** | rec | man | man | man | man |

### 制限
| 制限 | 状態 | 由来 | 出典 |
|---|---|---|---|
| ベジタリアン / ヴィーガン | 添え次第（プレーンなそばがきは可。つゆが魚を加える）| [INFERRED·med] → GAP |
| グルテンフリー | つなぎ小麦と醤油つけ汁がなければ**可**の可能性 | [UNKNOWN] → **GAP** |
| no-pork / no-beef / ハラル | おそらく可 | 肉なし | [INFERRED·med] → GAP |

### ここで生じた gap
- そばがきにつなぎ小麦が含まれるか → 小麦 / グルテンフリー（唯一の小麦不使用料理の
  可能性 —— 安全な推奨として確認する価値あり）
- 何と一緒に提供されるか（プレーン / 醤油つけ汁 / だしつゆ）→ 大豆/魚 + ベジタリアン

## 横断的な安全メモ
そばは**重篤でアナフィラキシー級**のアレルゲン。この店の全料理（おそらく厨房全体）が
そばに関わる。カレーうどん（小麦麺）でさえ、そば粉を扱う厨房で作られる →
そばアレルギーのお客様にとって**交差汚染**が現実のリスク。これ自体がオーナーへの
gap 質問（共用器具 / 共用の茹で湯）。
