## A. 安全に関わる（アレルゲン / 制限）—— コンシェルジュがアレルギー質問に答える前に必ず解消

1. **[OPEN]** **交差汚染（店全体）** —— この厨房は重篤アレルゲンのそば（buckwheat）粉を
   扱う。そばアレルギーのお客様に対し、いずれかの料理を安全に作れるか、それとも全料理で
   茹で湯/作業面/器具を共用しているか？
   a) 全料理がそば汚染の器具を共用（安全でないと想定）
   b) カレーうどんは完全に別調理
   c) その他 → ______
   *コンシェルジュがある料理を「そば安全」と言えるか否かを決定する。*

2. **[OPEN]** **麺の組成 — そば（料理1, 2）**: そば麺は
   a) 二八（小麦つなぎを約20%含む）、b) 十割（100% そば粉、小麦なし）、c) その他？
   *ざるそば・天ぷらそばの小麦アレルゲン + グルテンフリー状態を決定する。*

3. **[OPEN]** **天ぷらの具材 — 天ぷらそば（料理2）**: 上にのる天ぷらは何か？
   a) 海老（→ えび/甲殻類アレルゲン）、b) 野菜のみ（かき揚げ/野菜）、
   c) 混合、d) 日替わり → ______
   *えびは最上位アレルゲン。提供する標準の天ぷらそばについて確定が必要。*

4. **[OPEN]** **天ぷらの衣の卵 — 天ぷらそば（料理2）**: 天ぷらの衣に卵を含むか？
   a) はい（衣に卵）、b) いいえ（小麦粉＋水のみ）、c) 不明 → ______
   *卵アレルゲンを決定する。*

5. **[OPEN]** **揚げ油 — 天ぷらそば（料理2）**: 天ぷらを揚げる油は:
   a) 純植物油、b) **ごま（ごま油）**入りブレンド、c) その他 → ______
   *ごまアレルゲンを決定する。*

6. **[OPEN]** **カレーの肉 — カレーうどん（料理3）**: カレーに入る肉（あれば）は何か？
   a) なし / 野菜、b) 鶏、c) 豚、d) 牛、e) その他 → ______
   *豚/牛/鶏アレルゲン および no-pork / no-beef / ハラル の制限を決定する。*

7. **[OPEN]** **カレールウの乳 — カレーうどん（料理3）**: カレールウ/ソースに牛乳や
   乳製品（例: 脱脂粉乳、クリーム）を含むか？ a) はい、b) いいえ、c) 不明 → ______
   *乳アレルゲン + 乳不使用を決定する。*

8. **[OPEN]** **つゆ / だしのベース — 料理1, 2, 4（およびカレーつゆ、料理3）**:
   汁/つけ汁に使うのは:
   - 醤油？ a) はい（→ 大豆 + 通常は小麦）、b) いいえ、c) たまり/小麦不使用醤油 → ______
   - だし？ a) かつお（bonito → 魚）、b) 昆布のみ（魚なし）、c) 煮干し入りブレンド
     （niboshi → 魚）→ ______
   *ほとんどの料理で 大豆/小麦/魚 アレルゲンとベジタリアン/ヴィーガン可否を決定する。*

9. **[OPEN]** **そばがきの添え — そばがき（料理4）**: どう提供するか？
   a) プレーン、b) 醤油つけ、c) だしつゆ、d) わさび醤油 → ______
   また: そばがき自体に小麦を含むか？ a) いいえ（そば粉＋水のみ）、b) はい。
   *そばがきを小麦不使用 / ヴィーガン向けの推奨にできるかを決定する。*
